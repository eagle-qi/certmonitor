# 一、文档概述

本系统是面向企业**内外网 Web 资产治理 \+ SSL 证书全生命周期管控**的一体化平台，实现公网备案资产自动挖掘、内网网段定时 URL 探测、资产精细化多维台账管理、Excel 批量资产导入校验、证书自动采集过期预警、公网域名 / 内网 IP 免费 HTTPS 证书自助签发、完整用户账号体系（邮箱注册 \+ SSO 单点登录）、细粒度权限控制、全流程操作审计、多维度可视化统计分析，构建资产从**发现 \- 录入 \- 审核 \- 管理 \- 证书监控 \- 证书自助申请 \- 风险预警**的全闭环管控能力，解决企业资产台账混乱、证书过期故障频发、资产录入效率低、账号登录方式单一等核心问题。

# 二、系统建设目标

1. 支持输入企业备案公司名称，自动探测对公网备案域名资产，自动录入待确认资产池；

2. 支持内网 IP/IP 网段定时任务探测，自动发现 HTTP/HTTPS 内网 URL 资产并录入待确认资产；

3. 实现 URL 资产全维度业务、人员、项目、岗位关联精细化管理，补齐完整资产属性字段；

4. 提供资产 Excel 批量导入、在线模板下载、全字段必填 \+ 格式 \+ 重复强校验能力，提升资产录入效率；

5. 实现 HTTPS 证书自动采集、有效期监测、多阈值过期预警、证书全生命周期管理；

6. 对标开源 allinssl 能力，支持公网域名、内网 IP 自助申请免费 HTTPS 证书，自动入库绑定资产；

7. 搭建完整用户管理体系，支持管理员手动建号、用户邮箱自助注册、企业 SSO 单点登录三种账号接入方式；

8. 构建多角色细粒度权限体系，全流程操作日志审计，保障系统访问安全合规；

9. 支持站内消息、邮件、短信多渠道风险预警，实现证书过期、资产异常、任务故障主动推送；

10. 提供资产、证书、探测任务多维度可视化统计报表，支撑运维决策分析。

# 三、整体功能需求

## 3\.1 用户管理与身份认证模块

### 3\.1\.1 用户基础管理

1. 账号全生命周期管理：新增、编辑、启用 / 禁用、注销、密码重置、强制下线；

2. 用户字段：登录账号、姓名、登录邮箱、手机号、所属部门、归属角色、账号状态、创建时间、最后登录时间、注册来源；

3. 账号状态枚举：正常、禁用、密码锁定、已注销、过期；

4. 支持用户列表筛选、导出、批量启用 / 批量禁用、批量角色分配。

### 3\.1\.2 邮箱自助注册功能

1. 后台可配置**注册开关**、企业邮箱域名白名单，管控外部恶意注册；

2. 注册流程：填写邮箱 / 密码 / 姓名 / 部门→发送邮箱验证码→校验通过自动创建账号；

3. 校验规则：邮箱格式校验、密码复杂度控制、验证码时效 \+ 发送限流、同邮箱唯一注册；

4. 新注册账号默认分配只读基础角色，管理员可后续调整权限。

### 3\.1\.3 SSO 单点登录集成

1. 登录页双入口：账号密码登录 \+ 企业 SSO 单点登录，后台支持 SSO 功能启停；

2. 兼容主流协议：OAuth2\.0、OIDC、CAS 企业统一身份认证协议；

3. 登录逻辑：跳转企业认证中心→身份鉴权回调→存在账号直接免密登录，无账号自动创建影子账号；

4. 支持配置 SSO 默认角色，可限制 SSO 账号仅允许单点登录、禁止密码登录。

### 3\.1\.4 登录安全策略

1. 多登录方式兼容：账号密码、邮箱密码、邮箱验证码免密、SSO 单点登录；

2. 防护机制：密码错误次数超限自动锁定账号、异地登录异常提醒、会话超时自动登出；

3. 个人中心：用户自主修改密码、更新个人资料、绑定通知邮箱 / 手机号、查看个人操作日志。

### 3\.1\.5 角色权限与日志审计

1. 预设角色：超级管理员、资产管理员、证书管理员、项目负责人、普通查看员，细粒度菜单权限隔离；

2. 全量操作日志：登录、注册、资产操作、探测配置、证书申请、批量导入、权限变更全程留痕；

3. 日志字段：操作人、操作时间、操作 IP、操作内容、操作结果，支持查询与导出审计。

### 3\.2\.2 资产基础操作

支持单条新增、编辑、删除、状态变更、详情查看；
支持多维度组合筛选：URL、公司、项目、业务系统、归属岗位、负责人、资产状态、资产来源；
支持资产全字段 Excel/CSV 数据导出。

### 3\.2\.3 资产批量导入管理

1. 模板下载：页面提供一键下载标准 Excel 导入模板，标注必填红星、内置岗位枚举说明；

2. 导入校验规则：

    - 必填字段空值拦截；

    - URL、手机号、邮箱、IP、端口格式正则校验；

    - 协议、归属岗位枚举值非法拦截；

    - 基于 URL \+ 协议唯一键自动去重；

3. 导入结果反馈：展示成功 / 失败条数，失败行号 \+ 错误原因可视化，支持错误日志单独导出；

4. 导入数据默认进入**待确认资产**列表，等待管理员审核。

### 3\.2\.4 资产审核流程

所有来源资产（探测 / 导入 / 手动新增）统一进入待确认池；
管理员核验资产有效性、补全缺失字段；
审核通过转为**已确认**正式资产，审核不通过标记无效或驳回重编。

## 3\.3 公网备案资产自动探测模块

1. 支持单企业名称输入、批量企业名称导入两种探测触发方式；

2. 对接备案查询接口，自动获取企业名下备案域名、备案号、备案主体、解析 IP、备案时间；

3. 自动生成 HTTP/HTTPS 标准 URL，探测连通性、状态码、页面标题；

4. 系统自动存量去重，仅新增未知资产，探测结果自动录入待确认资产池。

## 3\.4 内网 URL 定时探测模块

1. 探测目标配置：单个内网 IP、自定义 IP 网段、自定义端口范围；

2. 协议选择：支持 HTTP、HTTPS、全协议探测；

3. 定时任务调度：支持分钟 / 小时 / 天 / 周周期配置，自定义探测执行时间段；

4. 安全管控：探测速率限流、IP / 端口黑白名单、内网数据隔离不对外传输；

5. 探测结果：新增 URL 录入待确认资产，存量资产自动更新在线状态与探测信息。

## 3\.5 SSL 证书全生命周期管理模块

1. 自动证书采集：对 HTTPS 资产自动抓取证书颁发机构、有效期、域名、加密算法、序列号、证书状态；

2. 过期监测预警：自定义 7/15/30/90 天多阈值预警规则；

3. 证书状态管理：正常、即将过期、已过期、无效、吊销；

4. 手动证书维护：支持离线证书信息录入、证书文件上传解析；

5. 资产双向关联：证书可反向查询所属 URL 资产，资产可快速查看绑定证书信息；

6. 证书操作留痕：续签、更新、替换全流程操作日志记录。

## 3\.6 免费 HTTPS 证书自助申请模块（对标 allinssl）

1. 支持两种申请场景：**公网域名**免费可信证书、**内网 IP**私有自签名证书；

2. 公网域名签发：支持 DNS 验证、HTTP 文件验证两种校验方式，异步后台签发；

3. 内网 IP 签发：无需公网校验，快速生成内网可用加密证书；

4. 证书能力：多格式打包下载（PEM、Nginx/Apache 部署包、PFX/P12）；

5. 自动入库绑定：签发成功证书自动写入系统证书库，一键关联对应 URL 资产；

6. 任务管理：展示申请中 / 校验中 / 签发成功 / 签发失败状态，支持日志查看、重新申请、证书下载；

7. 风控限制：申请频次限流、域名 / IP 黑白名单管控，私钥文件加密存储权限隔离。

## 3\.7 预警通知管理模块

1. 预警触发类型：证书即将过期、证书已过期、URL 资产无法访问、探测任务异常、证书申请签发失败；

2. 通知渠道：系统站内消息、邮件通知、短信通知；

3. 定向推送逻辑：自动匹配资产负责人、项目经理手机号与邮箱，精准推送责任到人；

4. 支持预警规则自定义、通知频次降噪，避免重复骚扰推送。

## 3\.8 统计报表可视化模块

1. 资产统计：资产总量、状态分布、协议占比、按公司 / 项目 / 业务系统 / 归属岗位 / 负责人多维度统计；

2. 证书统计：有效期分布、过期风险证书数量、颁发机构分布、证书来源分类（探测采集 / 手动录入 / 自助申请）；

3. 探测统计：定时任务执行次数、探测成功率、新增资产数量、探测异常明细；

4. 可视化展示：饼图、柱状图、列表多形式渲染，支持自定义条件报表导出。

# 四、非功能需求

## 4\.1 性能需求

1. 单内网 / 24 网段探测耗时≤10 分钟，单企业备案探测耗时≤5 分钟；

2. 页面加载≤3 秒，数据筛选查询响应≤2 秒；

3. 单次资产批量导入支持 1000 条以上，全量校验导入耗时≤30 秒；

4. 公网证书签发≤5 分钟，内网 IP 证书签发≤30 秒；

5. 邮箱验证码发送≤10 秒，SSO 登录回调认证≤3 秒；

6. 系统支持 100 \+ 并发在线用户，支撑 10 万 \+ 资产数据稳定存储。

## 4\.2 稳定性需求

1. 系统全年可用率≥99\.5%，7×24 小时不间断运行；

2. 探测任务、证书签发任务支持断点续跑，异常中断自动恢复；

3. 数据库定期自动备份，保障资产、证书、账号日志数据不丢失。

## 4\.3 安全性需求

1. 用户密码高强度加密存储，全程传输 HTTPS 加密；

2. 内网探测遵循安全规范，无恶意扫描、无敏感数据外泄；

3. 细粒度权限隔离，禁止越权查看 / 编辑资产、私钥、用户隐私信息；

4. SSO 对接通信加密，验证码限流防刷，账号登录异常风险告警。

## 4\.4 兼容性需求

1. 服务端支持 Windows、Linux 主流服务器部署；

2. 前端兼容 Chrome、Firefox、Edge 主流浏览器；

3. 批量导入支持标准 xlsx 格式，兼容主流备案接口、企业 SSO 协议。

# 五、核心业务流程

1. **公网备案资产探测流程**
管理员输入企业名称→调用备案接口获取域名→生成 URL 探测状态→自动去重→录入待确认资产→管理员审核→转为正式资产

2. **内网定时 URL 探测流程**
配置 IP 网段 \+ 探测规则 \+ 定时策略→后台异步扫描→发现 Web URL 资产→录入待确认资产→审核入库纳入台账

3. **资产批量导入流程**
下载标准导入模板→本地填写完整字段→上传文件→系统全维度校验→成功资产进入待确认→审核生效

4. **免费证书申请流程**
选择公网域名 / 内网 IP→提交签发申请→后台异步校验签发→证书自动入库绑定资产→多格式下载部署→纳入过期预警监控

5. **邮箱注册流程**
登录页点击邮箱注册→填写信息获取验证码→邮箱校验通过→自动创建账号→管理员分配权限

6. **SSO 单点登录流程**
点击 SSO 登录→跳转企业认证中心→鉴权回调系统→自动匹配 / 创建账号→免密进入系统

7. **证书过期预警流程**
HTTPS 资产自动采集证书→监测有效期达到阈值→多渠道定向推送预警→责任人更新证书→系统刷新证书状态
